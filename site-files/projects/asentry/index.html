Overview

This device uses a Raspberry Pi Pico W and CircuitPython to monitor NASA JPL’s Sentry service and warn of potential Earth-threatening asteroids.

If you want to do this on your computer instead of a stand-alone device, there’s a Python program for that.

Source files for this project are on GitHub.
