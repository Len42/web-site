Grounding Plug

Simple input grounding plug for my homebrew Eurorack synth

  • Back to Synth

This is something that you probably don't need for your Eurorack synth: An input grounding plug.

It's simply a patch plug with the signal (tip contact) connected to ground (sleeve contact), and a ground symbol painted on the side so I don't forget what it is.

The purpose of this grounding plug is to override the "normal" connection of an input jack, forcing the input to 0 volts (ground).
